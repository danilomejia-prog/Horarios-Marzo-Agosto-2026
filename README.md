PUBLICAR EL PORTAL EN GITHUB PAGES

1. Ingrese a https://github.com y cree una cuenta o inicie sesion.
2. Cree un repositorio nuevo, por ejemplo: horario-cia-2026
3. Suba el archivo index.html a la raiz del repositorio.
4. En el repositorio, vaya a Settings > Pages.
5. En Build and deployment, seleccione:
   - Source: Deploy from a branch
   - Branch: main
   - Folder: / (root)
6. Guarde los cambios.
7. Espere unos segundos y GitHub mostrara el enlace publico del sitio.

Ejemplo de enlace:
https://SU-USUARIO.github.io/horario-cia-2026/

RECOMENDACIONES
- Comparta el enlace web, no el archivo HTML por WhatsApp.
- Si actualiza el horario, reemplace el archivo index.html y GitHub actualizara la pagina.
- El portal funciona mejor en Chrome, Edge, Firefox y navegadores moviles modernos.

ARCHIVOS DEL PAQUETE
